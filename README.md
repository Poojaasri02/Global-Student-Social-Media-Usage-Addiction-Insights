# Global-Student-Social-Media-Usage-Addiction-Insights
Power BI Dashboard Project | Social Media Usage & Addiction Analysis
Excited to share my latest Power BI dashboard analyzing global student social media usage and its impact on mental health, academics, and lifestyle.
🔍 Key Insights:
• 📱 Social media usage strongly influences academic performance
• 🧠 Mental health varies with addiction levels
• 🌍 Country-wise trends show global patterns
• ⏳ Higher usage hours → increased addiction scores
• 👥 Gender & relationship status reveal behavioral differences
